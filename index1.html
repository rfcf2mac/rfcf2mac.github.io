Class Description

    Is a career in programming really for you? Spend a Saturday with us to build your first web page and test your new skills to see if you enjoy the prospects and process of software development. Find out how websites are built, and code a webpage yourself using industry-standard tools and professional coding practices.
    This class is designed to give you a look inside a 'day-in-the-life' of a software developer. Please read all of the requirements and set-up information to prepare for a great class.
